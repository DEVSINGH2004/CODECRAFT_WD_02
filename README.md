# ⏱️ Stylish Stopwatch App (HTML + Tailwind CSS + JavaScript)

A clean, responsive, and interactive **stopwatch web application** built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. It features start, pause, reset, and lap time functionality with a modern UI.

---

## 🌟 Features

- ✅ Real-time stopwatch with `HH:MM:SS` format
- ⏯️ Start, pause, and reset functionality
- 🏁 Record lap times with dynamic listing
- 🎨 Fully styled with **Tailwind CSS** and custom CSS
- 💻 Responsive design and accessible UI
- 📜 Built without any external JavaScript frameworks

---

## 🚀 Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
